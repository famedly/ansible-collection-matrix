# timetracking-bot

Deploys a dockerized
[timetracking-bot](https://gitlab.com/famedly/bots/timetracking) instance.

## Requirements

You need a postgres DB for it and docker needs to be installed. Matrix user must
also already exist.

## Role Variables

Look at `defaults/main.yml`.
